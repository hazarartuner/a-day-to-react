React Talks

React/Redux tecrübelerimizi aktarıyoruz

React ile uyguluma geliştirme aşamalarını merak ediyor musunuz, ya da bilgilerinizi tazelemek ister misiniz? Eğer bize ayıracak 1 gününüz varsa, size tecrübelerimizi aktarmak ve ilgi çekici konu başlıkları altında baştan sona react ile uygulama geliştirme süreçlerini kendi tarifimizle anlatmak isteriz.

React

React, Facebook ve bağımsız yazılımcılar tarafından geliştirilen bir "View" kütüphanesidir.

Routing

React uygulamalarında neredeyse standart olarak kabul edilen React-Router ile routing işlemlerimizi yapacağız.

Redux/Immutable.js

Redux, uygulama state'ini yönetmek için kullanılan bir kütüphanedir. Immutable.js kütüphanesi ile state'in mutate edilmemesini sağlayacağız.

SSR

React ile server side rendering yapmak artık o kadar da zor değil :) Peki biz nasıl yapıyoruz?
