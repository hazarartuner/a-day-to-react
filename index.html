1 Gün Boyunca React Konuşuyoruz

React/Redux tecrübelerimizi aktarıyoruz

React ile uyguluma geliştirme süreçlerini merak ediyor musun? Ya da bilgilerini tazelemek ister misin? Eğer bize ayıracak 1 günün varsa, sana tecrübelerimizi aktarmak ve ilgi çekici konu başlıkları altında baştan sona react ile uygulama geliştirme süreçlerini kendi tarifimizle anlatmak isteriz.

React

React, Facebook ve bağımsız yazılımcılar tarafından geliştirilen bir "View" kütüphanesidir.

Routing

React uygulamalarında neredeyse standart olarak kabul edilen React-Router ile routing işlemlerimizi yapıyoruz.

Redux/Immutable.js

Redux, uygulama state'ini yönetmek için kullanılan bir javascript kütüphanesidir. Immutable.js ile state'imizin mutate edilmemesini garanti altına alıyoruz.

SSR

React ile server side rendering yaparak Universal uygulamalar geliştirebiliyoruz.
